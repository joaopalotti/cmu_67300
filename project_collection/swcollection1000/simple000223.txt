If

disambig 

 If  is a word to describe a statement where one thing depends on something else. 

For example:
 We can call this true  if  there is proof.
 We will play outside  if  it does not rain.

 
 If --  is a poem written by  Rudyard Kipling . It appeared in the  Brother Square Toes  chapter of Kipling's book  Rewards and Fairies . In a  1995   BBC  opinion poll, it was voted  Britain Britain's  favourite poem. It is arguably Kipling's most famous poem.
